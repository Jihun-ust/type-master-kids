# Type Master Kids

A fun, interactive touch-typing tutor built specifically to help kids learn how to type! Type Master Kids guides young learners to master the keyboard through intuitive visual guidance, real-time feedback, and a structured, progressive curriculum.

## ✨ Features

- **Dynamic Visual Hands Guide**: A transparent hands overlay dynamically illustrates the exact finger and reaching gesture needed for every key press. This builds muscle memory and reinforces proper touch-typing habits so kids don't have to look down at their hands.
- **Interactive Virtual Keyboard**: An animated on-screen keyboard highlights the target key and provides a satisfying visual "ripple" upon correct presses.
- **Structured Curriculum**: Over 50 bite-sized levels grouped by row. Start with the Home Row, progress to the Top and Bottom rows, practice the Spacebar, and eventually master full-keyboard combinations.
- **Real-Time Feedback**: Immediate visual cues, including satisfying success states and engaging "shake" animations on mistakes, keep learners engaged and instantly correct errors.
- **Light & Dark Mode**: A built-in theme toggle to easily switch between a crisp, colorful light mode and a sleek, high-contrast dark mode.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jihun-ust/type-master-kids.git
   ```

2. Navigate into the project directory:
   ```bash
   cd type-master-kids
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173`).

## 🛠️ Tech Stack

- **React** (UI Framework)
- **Vite** (Build Tool)
- **Vanilla CSS** (Responsive layout, animations, and CSS variables for theming)

## 🎮 How to Play

1. Select a lesson from the scrollable curriculum menu on the right sidebar.
2. Watch the on-screen hands overlay to see exactly which finger to use and how to reach for the key.
3. Type the highlighted characters in the target sequence box.
4. Keep practicing to build speed and accuracy!
